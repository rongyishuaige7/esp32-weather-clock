# 硬件与接线边界

本文仅按当前公开源码记录接口与风险；它不是实物接线证明、PCB 原理图、电气审查或产品规格书。

## 源码接口表

| 模块/信号 | ESP32 接口 | 当前源码证据 | 边界 |
| :-- | :-- | :-- |
| DHT22 | GPIO4 | `DHT dht(DHT_PIN, DHT22)`；`DHT_PIN=4` | DHT22 的实际供电、上拉、线长、精度、响应和抗干扰均未按当前提交复测。 |
| I2C 总线 | SDA=GPIO21、SCL=GPIO22 | `Wire.begin(I2C_SDA, I2C_SCL)` | OLED、BH1750、DS3231 共用总线；上拉、电平与物理接线待确认。 |
| SSD1306 OLED | I2C `0x3C` | `display.begin(..., OLED_ADDR)` | 源码初始化失败会进入无限循环；地址、分辨率、供电和实物显示待复测。 |
| BH1750 | I2C `0x23` | `bh1750.begin(BH1750::CONTINUOUS_HIGH_RES_MODE)` | 实际模块地址、采样范围和读数未校准。 |
| DS3231 RTC | I2C `0x68` | `RTC_DS3231 rtc` | 失电时源码会设置固定默认时间；电池、走时和时区待确认。 |
| 配网 AP | Wi-Fi | `WiFi.softAP(AP_SSID)` | 是开放 AP，不是安全网络；只能短时用于隔离可信本地配置。 |
| EEPROM | ESP32 内部 Flash 模拟 EEPROM | `EEPROM.writeString` | 保存 Wi-Fi 和 API Key；不等于加密、安全存储或数据保护。 |

## 电源与安全注意事项

- ESP32 GPIO 为 3.3 V 逻辑；不要把 5 V 数字信号直接接入 GPIO；
- 所有低压模块需要可靠供电和共地；USB、OLED、RTC、传感器的实际电压范围须以实物资料为准；
- I2C 同一总线的地址、上拉、电容和线长会影响可靠性；不要将源码常量当作实际接线证明；
- 此仓库不包含电池、充电、外壳、过流保护、EMC、功耗、消防、环境耐受或安全认证设计；
- 不得把显示的温湿度、光照、时间或天气字段用于安全、医疗、生产控制或其他高后果决策。
